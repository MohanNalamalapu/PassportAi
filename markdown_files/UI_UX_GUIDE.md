# UI/UX Guide

Visual style guidelines and UI page layouts for PassportAI.

## Design Aesthetic
*   **Theme**: Dark, premium navy blue theme (`#1A1A2E` and `#0F0F1A`) combined with warm gold/cream accents (`#C9A96E`).
*   **Typography**: Headings use `Syne` for modern aesthetics; body text uses `DM Sans` for clean, professional reading.
*   **Layout**: Dynamic, responsive layouts that scale to mobile and desktop screens.

## Layout Flows

### 1. Homepage / Selector
*   Hero section presenting the before-and-after animated demo.
*   Simple country selector dropdown for template matching.

### 2. Upload Experience
*   Clean file upload card displaying dropzone states.
*   Processing timeline illustrating background removal and landmark progress.
*   Template summary panel displaying specific dimension requirements.

### 3. Results Screen
*   A side-by-side comparison comparing the source selfie with the final AI-centered cropped output.
*   Interactive compliance checklist card showing rules that passed or failed.
*   A download panel with buttons to save JPEG, PNG, and PDF formats.
